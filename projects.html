Projects

Portfolio Website

Built a portfolio website that is responsive and accessible using semantic HTML and ARIA roles.

  • Used the capabilities of HTML5 and CSS3 with a mobile first mentality to make the static site as responsive as possible.
  • Used the WAVE Accessibility Evaluation Tool and the Lighthouse Chrome extension to check my website.
  • Learned error handling with rendering elements onto the page as desired.
  • Click on the image to view the Lighthouse report for my Home page.

Tools used: HTML, CSS, Compiled SASS, Github, WAVE, Lighthouse, Canva, Netlify, and Visual Studio Code.

Weather App using API

Created a responsive weather app that accurately provides the weather of any city that is searched. Also has CRUD functionality when creating a weather dashboard of different cities. Working with react hooks and fetching data from an API turned out to be a challenge and that resulted in active learning. This project solidified how to pass props down to child components correctly.

  • The Home page uses geolocation to retrieve the local weather with a static google map.
  • The Dashboard is like a CRUD app but for weather... type in a city name to create a weather tile, then every tile can be deleted and/or have its weather updated.
  • The Map page just has an embedded interactive googlemap.

Tools used: HTML, CSS, JavaScript, React.js, JSX, Axios, Open Weather API, Google Maps API, Google Geolocation API, React-Bootstrap, and React-Router.

My Resource Bank

Created a chrome extension that banks the tab URL and stores the imformation in the browser's local storage until deleted. Includes an input field to label URL before banking. In this process, I learned how to take advantage of the browser local storage. I also learned about JSON manifest and the permissions allowed by Google Chrome.

  • Learned about JSON manifest.
  • Learned how to use local storage to hold to store the banked info until deleted.
  • Experienced the limitations and permissions required when developing a Google Chrome extension.

Tools used: HTML, CSS, JavaScript, Visual Studio Code, and Chrome Extensions Developer mode.
