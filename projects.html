Projects

My Portfolio Website

Created a portfolio website that is accessible using semantic HTML and ARIA roles. I used the WAVE Accessibility Evaluation Tool and the Lighthouse Chrome extension to check my website. When encountering errors and how to fix them, I expanded my knowledge into accessibility compliance and website best practices. I also learned what is possible when working within the limitations of HTML and CSS. Without any other libraries or frameworks, I was able to design a responsive and mobile device friendly website.

Click on the image to view the Lighthouse report for my Home page.

Tools used: HTML, CSS, Netlify, WAVE, Lighthouse, and Canva.

My Resource Bank

Created a chrome extension that banks the tab URL and stores the imformation in the browser's local storage until deleted. Includes an input field to label URL before banking. In this process, I learned how to take advantage of the browser local storage. I also learned about JSON manifest and the permissions allowed by Google Chrome.

Tools used: HTML, CSS, and JavaScript
